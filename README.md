## Pine Script Indicator

This repository now contains an advanced Pine Script indicator for TradingView with the following features:

### 📊 Advanced S/R Zones with Patterns & Alerts

**Files:**
- `advanced_sr_indicator.pine` - Main Pine Script indicator
- `README_PINE_INDICATOR.md` - Detailed documentation

**Key Features:**
- ✅ Optimalized S/R zone calculation with performance considerations
- ✅ Visualization using boxes or lines for S/R zones  
- ✅ BUY/SELL alerts functionality
- ✅ Extended patterns: Doji and Harami candlestick patterns
- ✅ Automatic cleanup of excess labels
- ✅ Timeframe selection for S/R zone calculations
- ✅ Statistics table with signal counts and estimated win-rate

**Quick Start:**
1. Copy the content of `advanced_sr_indicator.pine`
2. Open TradingView Pine Editor (Alt + E)
3. Paste and click "Add to Chart"
4. Configure settings as needed

See `README_PINE_INDICATOR.md` for full documentation.
